Race Overview

The Ragnar Relay is a unique endurance event where teams of runners cover a set distance over a 24-hour period. Each team member runs multiple legs, alternating with teammates in a van, creating a continuous relay race. The event fosters teamwork, camaraderie, and personal achievement, set against scenic backdrops.
